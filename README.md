# Templates for boilr

These templates are customized to fit my personal requirements.

# Usage

1. clone repo
2. cd $reponame
3. `./install_templates.sh`

# Update

1. `cd $clonedRepo`
2. `git pull origin`
3. `./install_templates.sh`
